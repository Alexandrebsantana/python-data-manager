# Python Data Manager

Meu primeiro projeto completo em Python, desenvolvido com o objetivo de praticar manipulação de dados, organização de código e persistência de informações utilizando arquivos CSV.

## Sobre o projeto

O sistema funciona como um pequeno gerenciador de dados em linha de comando, permitindo realizar operações de cadastro, consulta, atualização e remoção de registros.

Os dados são armazenados em um arquivo CSV, simulando um banco de dados simples para fins de aprendizado.

## Funcionalidades

* Listar registros cadastrados
* Buscar registros por nome
* Adicionar novos registros
* Atualizar quantidades
* Remover registros
* Gerar relatório com o valor total do estoque

## Tecnologias utilizadas

* Python
* Módulo CSV da biblioteca padrão

## Conceitos praticados

* Funções
* Listas e dicionários
* Manipulação de arquivos CSV
* Estruturas de repetição
* Estruturas condicionais
* CRUD (Create, Read, Update e Delete)
* Organização de código em módulos e funções
* Git e GitHub

## Como executar

1. Clone o repositório:

git clone https://github.com/Alexandrebsantana/python-data-manager.git

2. Acesse a pasta do projeto:

cd python-data-manager

3. Execute o programa:

python estoque.py
