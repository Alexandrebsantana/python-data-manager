- Listar registros
- Buscar registros
- Adicionar registros
- Atualizar registros
- Remover registros
- Gerar relatório

## Tecnologias

- CSV

## Conceitos praticados

- Funções
- Listas
- Dicionários
- CRUD
- Manipulação de arquivos
- Estruturas de repetição
- Estruturas condicionais
